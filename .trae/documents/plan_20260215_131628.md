# 继续添加经济指标

## 计划添加的指标

### 1. 利率数据
- 使用 `ak.macro_bank_china_interest_rate()` 获取中国央行利率
- 数据格式：日期、今值、预测值、前值

### 2. 汇率数据
- 使用 `ak.currency_boc_safe()` 获取人民币汇率
- 或使用 `ak.currency_latest()` 获取最新汇率

### 3. 更新后端
- 在 `data_fetcher.py` 添加 `fetch_rate_data()` 和 `fetch_exchange_data()` 方法
- 更新 `fetch_methods` 字典
- 更新 `get_available_indicators()` 列表

### 4. 更新前端
- 更新仪表盘显示更多指标卡片
- 更新经济指标页面列表
